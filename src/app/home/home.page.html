Led controller for Arduino

This is an app to control an RGB LED strip using Bluetooth.

The app connects to an Arduino with an HC-05 Bluetooth module and sends strings with commands. These commands are interpreted by the Arduino to control the LED. The Arduino must be programmed to interpret the commands.

The commands sent are:

Brightness : <brightness#80> , where the value is between 0 and 255.

Color: <setLedColorAll#128,128,128> , where the values are RGB components.
